# 📊 サンプルデータ説明書

## 概要

QC7 つ道具システムで使用できる 3 種類のサンプルデータを提供しています。それぞれ異なる特徴を持ち、様々な分析パターンを学習できます。

## 📁 サンプルデータ一覧

### 1. `sample_data.csv` - 基本サンプルデータ

- **データ期間**: 2024 年 1 月 1 日〜1 月 10 日（10 日間）
- **データ件数**: 40 件
- **特徴**: 基本的な不良パターンを学習
- **用途**: 初回学習・基本機能確認

### 2. `sample_data_detailed.csv` - 詳細サンプルデータ

- **データ期間**: 2024 年 1 月 1 日〜1 月 31 日（31 日間）
- **データ件数**: 155 件
- **特徴**: より現実的な工場データ
- **用途**: 本格的な分析練習

### 3. `sample_data_seasonal.csv` - 季節変動サンプルデータ

- **データ期間**: 2024 年 1 月 1 日〜2 月 29 日（60 日間）
- **データ件数**: 240 件
- **特徴**: 改善効果を表現（1 月は高不良率、2 月は改善）
- **用途**: 改善効果の分析・時系列分析

## 📋 データ項目説明

| 項目名   | 説明               | データ例                           |
| -------- | ------------------ | ---------------------------------- |
| 日付     | データの日付       | 2024-01-01                         |
| 製品名   | 製品の名称         | 製品 A, 製品 B, 製品 C, 製品 D     |
| 不良項目 | 不良の種類         | 寸法不良, 表面粗さ, 欠け, 傷, 変形 |
| 不良数   | 不良の件数         | 1〜11 件                           |
| 検査数   | 検査した件数       | 80〜120 件                         |
| 原因分類 | 不良の原因         | 加工, 工具, 材料, 作業者, 環境     |
| 発生工程 | 不良が発生した工程 | 旋盤, 研削, 組立, 熱処理           |
| 備考     | その他の情報       | 具体的な原因説明                   |

## 🎯 各サンプルデータの特徴

### 基本サンプルデータ（sample_data.csv）

```
特徴:
- シンプルな不良パターン
- 基本的なQC7つ道具の学習に最適
- データ量が少なく、処理が高速

学習ポイント:
- パレート図の基本読み方
- 特性要因図の4M分析
- 管理図の基本概念
```

### 詳細サンプルデータ（sample_data_detailed.csv）

```
特徴:
- より現実的な工場データ
- 複数の製品・工程を含む
- 詳細な原因分析が可能

学習ポイント:
- 製品別・工程別の比較分析
- 原因別の詳細分析
- より複雑な相関分析
```

### 季節変動サンプルデータ（sample_data_seasonal.csv）

```
特徴:
- 改善効果を表現（1月→2月で不良率低下）
- 時系列分析に最適
- 改善活動の効果測定が可能

学習ポイント:
- 時系列グラフでの改善効果確認
- 管理図での工程安定性評価
- 改善前後の比較分析
```

## 📊 分析例

### パレート図での分析

1. **基本データ**: 寸法不良が最多、表面粗さが 2 番目
2. **詳細データ**: 製品 A の寸法不良が突出
3. **季節変動データ**: 1 月は寸法不良、2 月は表面粗さが最多

### 特性要因図での分析

1. **加工原因**: 工具摩耗、治具不良
2. **工具原因**: 切削油不足、砥石摩耗
3. **材料原因**: 材料不良
4. **作業者原因**: 取扱不注意
5. **環境原因**: 温度変化、温度管理不良

### 管理図での分析

1. **基本データ**: 比較的安定した工程
2. **詳細データ**: ややばらつきのある工程
3. **季節変動データ**: 1 月は不安定、2 月は改善

## 🔧 使用方法

### 1. データのアップロード

1. サイドバーの「📁 データアップロード」から CSV ファイルを選択
2. または「📋 サンプルデータを生成」ボタンでテストデータを作成

### 2. 分析の進め方

1. **基本データ**で各ツールの基本操作を学習
2. **詳細データ**でより複雑な分析を練習
3. **季節変動データ**で改善効果の分析を実践

### 3. 比較分析

- 複数のサンプルデータを順番に分析
- 改善前後の効果を比較
- 異なる製品・工程の特徴を把握

## 💡 学習のコツ

### 段階的学習

1. **第 1 段階**: 基本データで各ツールの操作を覚える
2. **第 2 段階**: 詳細データで複雑な分析を練習
3. **第 3 段階**: 季節変動データで改善効果を分析

### 実践的な分析

1. **パレート図**: 重点管理項目の特定
2. **特性要因図**: 原因の体系的整理
3. **管理図**: 工程の安定性評価
4. **時系列グラフ**: 改善効果の確認

### 改善活動のシミュレーション

1. **現状把握**: 1 月のデータで問題点を特定
2. **原因分析**: 特性要因図で原因を整理
3. **対策実施**: 仮想的な改善活動を実施
4. **効果確認**: 2 月のデータで改善効果を確認

## 📈 期待される学習効果

### 基本スキル

- QC7 つ道具の基本的な使い方
- データの読み方と解釈
- グラフの作成と分析

### 応用スキル

- 複雑なデータの分析
- 改善活動の計画と実行
- 効果測定と評価

### 実践スキル

- 実際の工場データでの分析
- 改善提案の作成
- 品質管理の標準化

---

**💡 ヒント**: サンプルデータを順番に分析することで、QC7 つ道具の使い方を段階的に習得できます。実際の工場データでも同じ手順で分析を進めてください。
