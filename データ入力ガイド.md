# 📝 データ入力ガイド

## 🎯 CSV ファイル作成の効率的な方法

### 方法 1: システム内で直接入力（推奨）

1. サイドバーの「📝 手動データ入力」をクリック
2. フォームに必要事項を入力
3. 「データを追加」ボタンで追加
4. 複数件入力後「✅ データを確定」

### 方法 2: 一括データ入力

1. 「📊 一括データ入力」セクションを使用
2. 以下の形式でデータを入力：

```
日付,製品名,不良項目,不良数,検査数,原因分類,発生工程,備考
2024-01-01,製品A,寸法不良,5,100,加工,旋盤,工具摩耗
2024-01-02,製品B,表面粗さ,3,80,工具,研削,砥石摩耗
```

### 方法 3: Excel から変換

1. Excel でデータを入力
2. 「名前を付けて保存」→「CSV UTF-8（コンマ区切り）」
3. システムにアップロード

## 📋 データ項目の説明

| 項目     | 必須 | 説明               | 入力例     |
| -------- | ---- | ------------------ | ---------- |
| 日付     | ✅   | データの日付       | 2024-01-01 |
| 製品名   | ✅   | 製品の名称         | 製品 A     |
| 不良項目 | ✅   | 不良の種類         | 寸法不良   |
| 不良数   | ✅   | 不良の件数         | 5          |
| 検査数   | ✅   | 検査した件数       | 100        |
| 原因分類 | ⚠️   | 不良の原因         | 加工       |
| 発生工程 | ⚠️   | 不良が発生した工程 | 旋盤       |
| 備考     | ❌   | その他の情報       | 工具摩耗   |

## 🎯 入力のコツ

### 効率的な入力方法

1. **毎日定期的に入力**: 1 日の終わりにまとめて入力
2. **選択肢を活用**: ドロップダウンから選択して入力ミスを防止
3. **一括入力**: 複数件まとめて入力して効率化

### データの品質向上

1. **統一した用語**: 原因分類や工程名を統一
2. **詳細な備考**: 具体的な原因を備考欄に記録
3. **定期的な見直し**: データの整合性をチェック

## 📊 入力例

### 基本的な入力例

```
2024-01-01,製品A,寸法不良,5,100,加工,旋盤,工具摩耗
2024-01-01,製品A,表面粗さ,3,100,工具,旋盤,切削油不足
2024-01-01,製品B,欠け,2,80,材料,研削,材料不良
```

### 複雑な入力例

```
2024-01-01,製品A,寸法不良,5,100,加工,旋盤,工具摩耗による寸法オーバー
2024-01-01,製品A,表面粗さ,3,100,工具,旋盤,切削油不足で表面が粗い
2024-01-01,製品B,欠け,2,80,材料,研削,材料の欠陥による欠け
```

## 🔧 トラブルシューティング

### よくある問題

**Q: データが正しく表示されない**
A: 文字エンコーディングを UTF-8 に設定してください

**Q: 数値が正しく認識されない**
A: カンマ区切りで正確に入力されているか確認してください

**Q: 日付が正しく表示されない**
A: YYYY-MM-DD 形式で入力してください（例：2024-01-01）

### データ検証

- 不良数 ≤ 検査数 であることを確認
- 日付が正しい形式であることを確認
- 必須項目がすべて入力されていることを確認

---

**💡 ヒント**: 最初は少ないデータで試してから、本格的にデータ入力を行うことをお勧めします。
